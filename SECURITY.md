# Security Policy

## Supported Scope

This is a didactic project. Security fixes may be applied on a best-effort
basis, but there is no formal support window.

## Reporting a Vulnerability

If you identify a security issue, contact the maintainers listed in
`MAINTAINERS` before opening a public issue.

Please include:
- a clear description of the issue
- reproduction steps
- impact assessment
- any suggested mitigation

## Project Constraints

This repository is intentionally limited to publicly accessible content flows.
Security reports proposing features that bypass third-party protections,
authentication, monetization gates, or anti-bot systems are outside the scope
of the project and will not be accepted as feature requests.
