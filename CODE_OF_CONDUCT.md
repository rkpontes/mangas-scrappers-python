# Code of Conduct

## Our Commitment

This project is a didactic repository. Participation is expected to remain
respectful, constructive, and focused on learning, collaboration, and clear
technical discussion.

## Expected Behavior

- Be respectful in discussions, reviews, and issue reports.
- Prefer clear, factual, and actionable communication.
- Accept feedback in good faith.
- Keep contributions aligned with the documented scope of the project.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory language or exclusionary behavior.
- Spam, trolling, or intentionally disruptive conduct.
- Encouraging misuse of the project outside its documented scope.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict
participation when behavior violates this code of conduct.

## Reporting

Report issues to the maintainers listed in `MAINTAINERS`.
